DIE DEMOKI BOTKARTEN
sorgen für den Start eigene Karten zu erstellen. Sie haben unterschiedliche Funktionen:
Der Lügenbaron:
Kopiert man den Text hinter dem QR-Code und fügt ihn in eine Chat-KI ein, antwortet der Bot immer genau mit dem Gegenteil beziehungsweise mit einer ganz falschen Antwort auf die ihm gestellte Frage oder Anweisung.
Der Essenskritiker:
Kopiert man seinen Text hinter dem QR-Code und fügt ihn in eine Chat-KI ein, kann man ihm sein eigenes Lieblingsessen vorstellen. Seine Antwort hat immer einen besonderen Twist. Er ist nicht ganz objektiv.
Der Grundgesetz Bot:
Er versucht (nach einfügen des Textes hinter dem QR-Code in eine Chat-KI) immer mit Bezug auf das deutsche Grundgesetz zu antworten. Hier ist auf Halluzinationen zu achten. Diese können im Unterrichtsgespräch besprochen werden.
Die Bots verfolgen das Ziel sich im Anschluss über die Erfahrungen mit Ihnen auszutauschen. Sie animieren zusätzlich dazu eigene Bots zu erstellen. Diese können ebenfalls in Kartenform gebracht und so ein ganzes Kartenspiel generiert werden. Eine Vorlage findet ihr hier im Download.
